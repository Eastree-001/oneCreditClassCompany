# 一学分课堂企业端

一个现代化的企业端平台，帮助企业解决人才培养、课程匹配和合作项目管理的痛点。

## 项目简介

一学分课堂企业端旨在解决以下企业痛点：
- 许多新兴岗位（AI算法工程师、AIGC内容设计师、数据安全分析师等）在高校中没有对应课程或专业
- 企业希望参与人才培养，却缺乏稳定、制度化的合作渠道
- 招聘后培训周期长，增加企业用人成本

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Element Plus** - 基于 Vue 3 的组件库
- **ECharts** - 强大的数据可视化库
- **Vue Router** - Vue.js 官方路由管理器
- **Pinia** - Vue 的状态管理库

## 功能特性

### 核心功能

1. **岗位技能画像管理**
   - 上传和管理岗位技能画像
   - 支持多技能标签
   - 岗位画像详情查看和编辑

2. **课程智能匹配**
   - 根据岗位画像智能匹配相关课程
   - 显示匹配度和匹配技能
   - 提供优化建议
   - 可视化匹配度分析

3. **人才需求发布**
   - 发布企业人才需求
   - 指定合作高校
   - 需求状态管理

4. **合作项目管理**
   - 管理与企业合作的人才培养项目
   - 项目进度跟踪
   - 项目类型分类（人才培养、课程共建、实习基地、科研合作）

5. **培训计划管理**
   - 创建和管理培训计划
   - 选择匹配的课程
   - 培训进度跟踪
   - 缩短培训周期

6. **数据分析看板**
   - 数据统计卡片
   - 课程匹配趋势图表
   - 岗位画像分布
   - 培训计划完成情况
   - 合作高校统计

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## UI 设计特点

- **现代化设计**：采用渐变色彩和卡片式布局
- **响应式设计**：适配不同屏幕尺寸
- **流畅动画**：页面切换和交互动画
- **数据可视化**：使用 ECharts 展示数据图表
- **用户体验优化**：清晰的导航和操作流程
